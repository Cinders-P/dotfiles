**Dotfiles**

Backup for personal use.

![Starship Prompt 'Rose Pine' Theme](./img/screenshot.png)

Installs:
- tools:
    - fzf (fuzzy-finder)
    - zoxide (autojump)
    - fd (find), ag/rg (grep), bat (cat)
    - duckDB, qsv (csv parsing)
    - mise (package manager)
    - tldr (help pages), shellcheck
    - python3, uv (dependency manager)
- bash plugins:
    - bash-completion (autocompletions)
    - Starship (terminal prompt)
- zsh plugins:
    - zsh-autosuggestions (show incremental history match)
    - zsh-syntax-highlighting (highlight commands)
    - starship (terminal prompt)
- vim plugins:
    - fzf.vim
    - commentary
    - vim-signature
- .bashrc, .vimrc, .inputrc, .zshrc, .ssh/config, starship.toml, fastfetch.jsonc files
- bonus: cartograph cf and monolisa nerd fonts
    - enable 'ss02' stylistic set in ligature settings, for script characters
